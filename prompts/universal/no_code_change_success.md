# Universal No Code Change Success Prompt

코드 변경이 필요 없을 때 작업을 무의미하게 늘리지 않고 바로 종결하기 위한 프롬프트입니다.

```markdown
검사 및 진단 결과, 요청된 버그나 문제 현상이 이미 수정 완료되어 있거나 신규 구현 없이 정상 동작함이 명백히 입증되었습니다.
억지로 코드를 고쳐서 일하는 척하지 마십시오 (Anti-pattern).
`system/verification_log.md`에 문제 미발생 증적을 기록하고 "변경 없음 성공(No Code Change Success)"으로 보고하고 작업을 종결하십시오.
```
