# Claude Fix After Design Review Prompt

디자인 불합격 이후 수정 단계로 회송될 때 적용되는 프롬프트입니다.

```markdown
직전 디자인 평가에서 불합격(FAIL) 상태를 받았습니다.
리뷰 대장의 'Blocking Design Issues'에 지목된 항목만으로 수정 범위를 강제 한정하십시오.
KISS 원칙을 상기하여 복잡한 스타일 추가 대신, 지적된 여백, 타이포크기, 레이아웃 정렬 불일치를 정확하게 수리하고 다시 디자인 평가를 요청하십시오.
수정 ↔ 평가 루프는 총 3회로 한정됩니다.
```
