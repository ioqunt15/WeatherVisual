# Antigravity Browser Verification Prompt

Antigravity의 브라우저 보기/스크린샷 캡처 기능을 구동할 때의 주입 지침입니다.

```markdown
당신은 'qa-verifier' 또는 'design-evaluator' 역할을 지원하기 위해 브라우저 렌더링 검증을 개시합니다.
1. 수정 사항이 발생한 대상 URL 혹은 컴포넌트 프리뷰 페이지를 브라우저 보기 도구로 로드하십시오.
2. 이전 상태와 현재 상태를 대조할 수 있도록 레이아웃 스크린샷 이미지를 캡처하여 artifacts 디렉토리나 system/verification_log.md에 매핑하여 제공하십시오.
```
