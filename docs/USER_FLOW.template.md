# User Flow (사용자 동선 설계서)

* **사용 목적**: 사용자가 특정 목적을 달성하는 경로를 설계하여 막다른 길(Dead End)이 생기는 것을 막습니다.
* **작성자**: UX Architect
* **포함 사항**: 플로우 맵(시나리오별 순서), 진입점(Entry Point), 이탈점(Exit Point), 중간 분기점.
* **제외 사항**: 디자인 컴포넌트 세부 스타일값.
